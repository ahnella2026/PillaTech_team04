## 🛠️ 협업 가이드 
4조 팀원분들은 `git pull`을 받은 후, 아래 순서대로 실행하여 학습 환경을 구축해 주세요.

### 1. 가상환경 및 라이브러리 설치
네이티브 환경 대신 `conda`나 `venv` 가상환경 사용을 권장합니다.

```bash
# 가상환경 생성 (예: pilla)
conda create -n pilla python=3.10 -y
conda activate pilla

# 정밀 버전 라이브러리 설치
pip install -r requirements.txt
```

### 2. 데이터셋 배치
원본 데이터는 용량 문제로 Git에 포함되지 않았습니다. **프로젝트 루트**에 아래 경로로 배치해 주세요.

- `./train_images/*.png` : 훈련 이미지 (232장)
- `./train_annotations/**/*.json` : 훈련 어노테이션 (763개)

### 3. 환경 변수 설정
루트의 `.env` 파일에 전처리 설정이 되어 있습니다. (기본 3-Seed 설정 완료)
수정이 필요한 경우 `.env` 파일의 `RANDOM_SEEDS` 등을 조정하세요.

### 4. 전처리 파이프라인 가동 
아래 명령어 한 줄이면 **Invalid BBox 필터링 + 3가지 시드(42, 123, 777) 기반 데이터셋 분할**이 자동으로 완료됩니다.

```bash
python src/data/coco_to_yolo.py
```

- 실행 후 `data/yolo/seed_42/dataset.yaml` 등 총 3개의 학습 세트가 생성됩니다.

## 🧪 전처리 핵심 전략 

- **Invalid BBox 제거:** 이미지 범위를 벗어난 비정상 좌표(`x=6567` 등)를 자동 필터링합니다.
- **3-Seed 교차 검증:** 클래스 불균형 대응을 위해 3가지 랜덤 시드로 독립적인 데이터셋을 생성합니다.
- **해상도 가이드:** EDA 결과(BBox 면적 5.6%)에 따라 **1024px 또는 1280px** 학습을 권장합니다. (640px은 실험용)
- **Flip 증강 금지:** 각인(글자/로고) 왜곡 방지를 위해 좌우 반전은 배제하고 Rotation(회전) 위주로 학습합니다.

### 5. Baseline 학습 시작 (원터치 스크립트)
복잡한 옵션 없이 아래 명령어로 즉시 최적화된 학습을 시작할 수 있습니다. 

```bash
# EDA 기반 Flip 금지/회전/해상도 설정이 반영된 학습 스크립트
python src/train_v8.py
```
- 내부적으로 `imgsz=1024`, `batch=16`, `Flip=OFF`, `Rotation=360` 등이 자동 적용됩니다.
